# bubblesheets-web

## About

This is a web verison of the Bubblesheets editor from the Bubblesheets app. https://github.com/sprucepine/bubblesheets 

## GitHub Pages

The frontend is deployed with GitHub Actions from the `main` branch.

- Workflow: `.github/workflows/deploy.yml`
- Build output: `frontend/dist`
- Live site: https://sprucepine.github.io/python-honors-project/

If the page looks blank, check the latest workflow run in the GitHub Actions tab and confirm the browser console does not show missing asset errors.
